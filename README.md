# riddle-me

Custom ZSH plugin to generate random Riddles.

## How to use?

1. Clone the repo into your custom plugins folder, usually at ($HOME/.oh-my-zsh/custom/plugins/)
2. Add `riddle-me` to .zshrc plugin list, usually at (~/.zshrc)

```zsh
plugins=(... archlinux)
```
